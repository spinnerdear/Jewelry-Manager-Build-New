# Jewelry Media Manager (Legacy v1.9)

โปรแกรมจัดการและจัดระเบียบไฟล์สื่อสำหรับสต็อกสินค้าเครื่องประดับ ออกแบบมาเพื่อลดขั้นตอนการทำงาน Manual ในการคัดแยกประเภทและตั้งชื่อไฟล์

## 🚀 คุณสมบัติเด่น (Features)
- **Automatic Grouping:** รวมไฟล์ที่มีรหัสเลข 4 หลักเหมือนกันเข้าโฟลเดอร์เดียวกันให้อัตโนมัติ
- **Visual Primary Photo Selection:** หน้าจอแกลเลอรี่ให้ผู้ใช้เลือกรูปภาพหลัก (Primary) เพื่อใช้เป็นชื่อไฟล์หลัก (เช่น `R-1234.jpg`) และรูปอื่นๆ จะถูกรันเลขต่อท้าย (เช่น `R-1234-2.jpg`)
- **Auto-Detect Downloads:** ตรวจหาโฟลเดอร์งานล่าสุดในเครื่อง (โฟลเดอร์ที่ชื่อขึ้นต้นด้วย "Media -") และถามเพื่อโหลดเข้าโปรแกรมทันที
- **Multi-Location Sync:** ย้ายและสำรองไฟล์ไปยัง 2 ตำแหน่งพร้อมกัน (เช่น Main Database และ Backup Drive)
- **Automated Archiving:** ย้ายโฟลเดอร์ที่ทำเสร็จแล้วไปยัง Archive Drive โดยแยกตามโครงสร้าง `ปี / เดือน / วัน`
- **Drag & Drop Support:** รองรับการลากโฟลเดอร์จากหน้าจอคอมพิวเตอร์มาวางในโปรแกรมได้เลย

## 🛠 การติดตั้ง (Installation)
สำหรับรันผ่าน Python สคริปต์:
1. ติดตั้ง Python 3.10 ขึ้นไป
2. ติดตั้งไลบรารีที่จำเป็น:
   ```bash
   pip install pillow tkinterdnd2
   ```

## 📖 คู่มือการใช้งาน (How to Use)
### 1. การตั้งค่าเริ่มต้น (Configuration)
- **PHOTO 1 & 2:** เลือกโฟลเดอร์ปลายทางหลักและสำรองที่ต้องการให้รูปไปเก็บ
- **ARCHIVE DRIVE:** เลือกโฟลเดอร์สำหรับเก็บประวัติงานที่ทำเสร็จแล้ว
- **MANAGE CATEGORIES:** ตรวจสอบตัวย่อรหัส (เช่น R=Ring, N=Necklace) ให้ตรงตามโครงสร้างฐานข้อมูล

### 2. ขั้นตอนการประมวลผล (Workflow)
1. **GROUP BY CODE:** รวบรวมไฟล์ดิบเข้าเป็นหมวดหมู่ตามเลขรหัส
2. **RENAME FILES:** (สำคัญ) หน้าจอแกลเลอรี่จะเปิดขึ้นมา ให้คุณคลิกเลือก **"รูปที่สวยที่สุด/ดีที่สุด"** เพื่อเป็นรูปหลัก
3. **COLLECT PHOTOS:** โปรแกรมจะนำรูปที่ Rename แล้วไปวางในฐานข้อมูลตามหมวดหมู่และช่วงรหัส 200 (เช่น Ring 10201-10400)
4. **MOVE TO ARCHIVE:** ย้ายโฟลเดอร์งานออกจากหน้าจอหลักเพื่อความสะอาดและเก็บเป็นประวัติ

## 📂 โครงสร้างฐานข้อมูลที่โปรแกรมรองรับ
โปรแกรมจะย้ายไฟล์ไปตามตำแหน่ง:
`Database Drive` > `{หมวดหมู่}` > `{หมวดหมู่} {ช่วงรหัส}` > `{ชื่อไฟล์}.jpg`
