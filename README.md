# Jewelry Media Manager (Legacy)

โปรแกรมจัดการไฟล์สื่อและรูปภาพสินค้าสำหรับเครื่องประดับ (เวอร์ชัน 1.9) เน้นการจัดระเบียบไฟล์และการคัดแยกประเภทสินค้าอัตโนมัติ

## 🌟 คุณสมบัติหลัก (v1.9)
*   **Drag & Drop Support:** รองรับการลากไฟล์มาวางเพื่อเริ่มกระบวนการจัดการได้ทันที
*   **Auto-Categorization:** คัดแยกประเภทสินค้า (Ring, Necklace, Earring, etc.) อัตโนมัติจากรหัสสินค้า
*   **Dual Photo Locations:** รองรับการจัดการรูปภาพพร้อมกันใน 2 ตำแหน่งหลัก
*   **Archive System:** ระบบสำรองไฟล์และเก็บข้อมูลประวัติการทำงาน
*   **Smart Detection:** ตรวจหาไฟล์ใหม่จากโฟลเดอร์ดาวน์โหลดโดยอัตโนมัติ
*   **Dark Theme GUI:** หน้าจอสวยงาม ใช้งานง่าย สบายตา

## 📂 โครงสร้างหมวดหมู่
โปรแกรมคัดแยกตาม Prefix รหัสสินค้า:
- `R`: Ring
- `N`: Necklace
- `E`: Earring
- `P`: Pendant
- `B`: Bracelet
- `S`: Sets

## 🛠 วิธีใช้งาน
1.  กำหนดตำแหน่งโฟลเดอร์รูปภาพ (Photo 1, Photo 2) และโฟลเดอร์สำหรับ Archive
2.  ลากไฟล์รูปภาพมาวางในช่อง Source หรือเลือกโฟลเดอร์ที่ต้องการจัดการ
3.  กดเริ่มทำงาน โปรแกรมจะทำการคัดแยกและย้ายไฟล์ไปยังโฟลเดอร์ตามหมวดหมู่และรหัสสินค้าให้อัตโนมัติ
4.  ตรวจสอบสถานะการทำงานได้จาก Activity Log และไฟล์ History
