# Jewelry Media Manager (Pro Cloud AI Edition)

เครื่องมืออัจฉริยะระดับมืออาชีพสำหรับจัดการและรีทัชรูปภาพสินค้าเครื่องประดับ (Kh Creation Standards) ขับเคลื่อนด้วยพลังของ **Gemini AI Cloud Agent**

## 🌟 คุณสมบัติเด่น (v2.0 Beta 19)

### 🤖 ระบบ Cloud AI Agent (ขั้นตอนที่ 1.5)
เปลี่ยนรูปถ่ายดิบให้กลายเป็นรูปสตูดิโอพรีเมียมโดยอัตโนมัติ:
- **Smart Background Removal:** ตัดพื้นหลังขาวบริสุทธิ์ (Pure White) เนียนกริบถึงระดับรูเล็กๆ
- **Shank Sharpener (Ring):** วาดและเพิ่มความคมชัดให้กับก้านแหวนที่เบลอจากการถ่ายมาโคร
- **Auto-Hanger Removal (Earring):** ลบอุปกรณ์ช่วยห้อยต่างหูออก และเจนส่วนก้านหูขึ้นมาทดแทน
- **Automatic Montage (Earring):** ตรวจจับรูปหน้า-ข้าง และรวมเป็นรูปเดียวอัตโนมัติ (ซ้าย:หน้า / ขวา:ข้าง)
- **AI Processing Status:** เพิ่มระบบแจ้งเตือนสถานะการทำงานของ AI แบบ Real-time เพื่อไม่ให้โปรแกรมดูเหมือนค้าง

### 🛠 ฟังก์ชันการจัดการไฟล์
- **Drag & Drop:** ลากโฟลเดอร์งานมาวางเพื่อเริ่มงานได้ทันที
- **Visual Primary Selection:** หน้าจอแกลเลอรี่แบบ Compact (v2.0) ให้คุณเลือกรูปที่รีทัชแล้วเป็นรูปหลัก
- **UI Label Restoration:** กู้คืนป้ายชื่อคำอธิบายไดรฟ์ให้ชัดเจน เข้าใจง่าย
- **Centralized Error Codes:** ระบบแจ้งเตือนปัญหา (E001-E007) พร้อมคำอธิบายภาษาไทย เพื่อการแก้ไขที่รวดเร็ว
- **Unified Sync:** ก๊อปปี้รูปเข้าฐานข้อมูลหลักและสำรองพร้อมกันในครั้งเดียว

## 📖 คู่มือการใช้งาน (Workflow)

1.  **GROUP BY CODE:** จัดกลุ่มไฟล์ดิบตามเลขรหัส 4 หลัก
2.  **CLOUD AI RETOUCH (1.5):** 🤖 ส่งรูปขึ้นไปให้ Gemini Agent จัดการรีทัชตามกฎ Kh Creation
3.  **RENAME FILES:** เลือกรูปภาพหลักจากชุดภาพที่รีทัชแล้วผ่านหน้าจอ Visual Selection
4.  **COLLECT PHOTOS:** โปรแกรมจะนำรูปที่สวยงามไปเก็บในฐานข้อมูลตามช่วงรหัส 200 (เช่น Ring 10201-10400)
5.  **MOVE TO ARCHIVE:** ย้ายงานที่เสร็จแล้วเข้าคลังประวัติ

## ⚙️ การตั้งค่าระบบ
- **API Key:** ตั้งค่าได้ผ่าน environment variable `GOOGLE_API_KEY` หรือกรอกในช่อง Google Cloud API Key ในโปรแกรม ห้าม commit key จริงลง repo
- **Path Config:** รองรับไดรฟ์เครือข่าย (เช่น X:, Z:) พร้อมระบบเช็คสถานะการเชื่อมต่ออัตโนมัติ

## 🛠 ความต้องการของระบบ (สำหรับการพัฒนา)
- **Python 3.10+**
- **Libraries:** ดูรายการเต็มใน `requirements.txt`
  ```bash
  pip install -r requirements.txt
  ```
