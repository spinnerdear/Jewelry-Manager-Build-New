# PixUp (Pro Cloud AI Edition)

เครื่องมืออัจฉริยะระดับมืออาชีพสำหรับจัดการและรีทัชรูปภาพสินค้าเครื่องประดับ (Kh Creation Standards) ขับเคลื่อนด้วยพลังของ **Gemini AI Cloud Agent**

## 🌟 คุณสมบัติเด่น (v2.1 Beta 1)

### 🤖 ระบบ Cloud AI Agent (ขั้นตอนที่ 1.5)
เปลี่ยนรูปถ่ายดิบให้กลายเป็นรูปสตูดิโอพรีเมียมโดยอัตโนมัติ:
- **Smart Background Removal:** ตัดพื้นหลังขาวบริสุทธิ์ (Pure White) เนียนกริบถึงระดับรูเล็กๆ
- **AI Processing Status:** ระบบแจ้งเตือนสถานะการทำงานของ AI แบบ Real-time และระบบจัดการ Error อย่างเข้มงวด

### ✂️ ระบบ Interactive Tools (ใหม่ใน v2.1)
- **1.6 Interactive Earring Merge:** รวมรูปต่างหูหน้า-ข้าง แบบปรับขนาด (Scale) และสลับซ้ายขวาได้อิสระบน Canvas 2000x2000 px
- **1.7 Smart Crop & Position:** จัดตำแหน่งและซูมรูปสินค้าบน Canvas 2000x2000 px แบบ Interactive เพื่อให้ได้องค์ประกอบที่สวยที่สุด

### 🛠 ฟังก์ชันการจัดการไฟล์
- **Drag & Drop:** ลากโฟลเดอร์งานมาวางเพื่อเริ่มงานได้ทันที
- **Visual Primary Selection:** หน้าจอแกลเลอรี่ให้คุณเลือกรูปหลักได้ง่ายๆ
- **Centralized Error Codes:** ระบบแจ้งเตือนปัญหา (E001-E999) พร้อมคำอธิบายภาษาไทย
- **Unified Sync:** ก๊อปปี้รูปเข้าฐานข้อมูลหลักและสำรองพร้อมกันในครั้งเดียว

## 📖 คู่มือการใช้งาน (Workflow)

1.  **GROUP BY CODE:** จัดกลุ่มไฟล์ดิบตามเลขรหัส 4 หลัก
2.  **CLOUD AI RETOUCH (1.5):** 🤖 เลือกรูปและส่งขึ้นไปให้ Gemini Agent จัดการรีทัช
3.  **MERGE EARRING (1.6):** 🔗 รวมรูปต่างหูแบบปรับแต่งได้เอง (เฉพาะต่างหู)
4.  **CROP & POSITION (1.7):** ✂️ จัดตำแหน่งและซูมรูปสินค้าให้สวยงาม (ยกเว้นต่างหู)
5.  **RENAME FILES:** เลือกรูปภาพหลักและเปลี่ยนชื่อไฟล์ตามมาตรฐาน
6.  **COLLECT PHOTOS:** โปรแกรมจะนำรูปที่สวยงามไปเก็บในฐานข้อมูลตามช่วงรหัส
7.  **MOVE TO ARCHIVE:** ย้ายงานที่เสร็จแล้วเข้าคลังประวัติ

## ⚙️ การตั้งค่าระบบ
- **API Key:** กรอกในช่อง Google Cloud API Key ในโปรแกรม หรือใช้ `GOOGLE_API_KEY`
- **Path Config:** รองรับไดรฟ์เครือข่ายและระบบเช็คสถานะการเชื่อมต่อ
